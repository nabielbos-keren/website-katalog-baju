# Archive Lab — Official Store

A static e-commerce storefront for Archive Lab, an Indonesian fashion/accessories brand.

## Stack

- Pure HTML + CSS + JavaScript (no build tools, no framework)
- Tailwind CSS via CDN
- WhatsApp-based checkout flow

## How to run

```
python3 -m http.server 5000
```

The workflow **Start application** is already configured for this. Open the preview pane to see the store.

## Features

- Product catalog with search and category filters
- Product detail page (size selector, quantity picker)
- Shopping cart (localStorage-persisted, slide-in drawer)
- Bulk order checkout via WhatsApp (`wa.me/6281217538688`)
- WhatsApp CS chat button (bottom-right)
